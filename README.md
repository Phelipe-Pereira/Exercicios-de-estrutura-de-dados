# 📚 Exercícios de Estrutura de Dados

Repositório dedicado à prática e estudo de **Estruturas de Dados e Algoritmos**, contendo implementações e exercícios desenvolvidos ao longo da minha formação em Análise e Desenvolvimento de Sistemas.

O objetivo deste projeto é consolidar conceitos fundamentais de lógica, desempenho e organização de dados, servindo como base para resolução de problemas mais complexos no desenvolvimento de software.

---

## 🚀 Conteúdos abordados

Neste repositório você encontrará implementações de:

- Vetores (Arrays)
- Listas encadeadas
- Pilhas (Stack)
- Filas (Queue)
- Algoritmos de ordenação
- Algoritmos de busca
- Recursividade
- Manipulação de dados

---

## 🧠 Objetivo

Este repositório tem como foco:

- Praticar conceitos fundamentais de estruturas de dados
- Melhorar raciocínio lógico e resolução de problemas
- Entender complexidade de algoritmos (tempo e espaço)
- Criar uma base sólida para entrevistas técnicas

---


## ▶️ Como executar

1. Clone o repositório:
git clone https://github.com/Phelipe-Pereira/Exercicios-de-estrutura-de-dados.git

2. Acesse a pasta:
cd Exercicios-de-estrutura-de-dados

3. Execute os arquivos conforme a linguagem utilizada.
